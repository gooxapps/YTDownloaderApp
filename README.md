# YTDownloaderApp

A fast, modern YouTube Downloader built with C# and Avalonia UI. 

## Current Features
- Fast video search using YoutubeExplode.
- Direct downloads via yt-dlp & ffmpeg.
- Format selection (4K, 1080p, 720p, 480p, Audio Only).
- Download queuing, pausing, and resuming.
- Velopack Auto-Updater integrated.

---

## Strategic Roadmap & Future Monetization Plan 💰

To ensure we build a loyal user base without sacrificing future revenue, we have agreed on the following strategic launch plan:

### Phase 1: Free Launch (Current Phase)
The app will be released to the public **100% Free**.
- **Goal:** Build a massive user base, get feedback, squash bugs, and establish trust.
- **Why:** To make the app a staple on users' machines and spread via word-of-mouth.

### Phase 2: Introduction of the "Freemium" Model
Once the user base is large and active, we will release a major update via the Velopack Auto-Updater that introduces "Pro" features.

**Crucial Agreement on the Transition:**
To avoid the "bait and switch" trap and prevent angry users, we will strictly follow the **"Charge for NEW Features"** approach.
1. **Existing Free Features Stay Free:** Anything that was free in V1.0 (e.g., 1080p downloads, single video extraction) will *remain free forever*.
2. **Charge for the New Stuff:** We will only put *brand new*, highly-requested features behind the Pro paywall. Examples of future Pro features could include:
   - Bulk Playlist Downloading.
   - 8K Video Extraction.
   - Advanced Custom File Naming Templates.

**Optional (The Grandfather Clause):** 
If technically feasible, we may choose to automatically unlock the "Pro" version for all users who installed the app during Phase 1, rewarding them for their early support while charging all new users going forward.
